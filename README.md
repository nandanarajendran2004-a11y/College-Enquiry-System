# College Enquiry System

The College Enquiry Site is a web application that enables students and external users to submit admission, general, and department-wise enquiries. Admins handle general and admission queries, HODs handle department queries, and a feedback system ensures efficient, responsive, and organized communication.

## Features
- Submit admission enquiries
- Submit department-wise enquiries
- Student and faculty management
- Admin and HOD dashboard for managing queries
- Feedback system for responsive communication

## How to Run
1. Install **WAMP** and start Apache & MySQL.
2. Place the project folder in the `www` directory:
C:\wamp64\www\miniproject
3. Open **phpMyAdmin** and import `enquiry.sql` to create the database.
4. Open your browser and navigate to:
http://localhost/miniproject
5. Login using default users (if provided) or register as a new user.
